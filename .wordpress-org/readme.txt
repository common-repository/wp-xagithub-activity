== Description ==

Show your github activity in a widget

Features:

*   Choose the number of entries to show
*   Style included

Follow the project in <a href="https://github.com/xaviermartinezf/WP-Xagithub-Activity">Github</a>

== Installation ==

How to install it.

1. Upload `wp-xagithub` folder to the `/wp-content/plugins/` directory.
2. Active `WP Xagithub Activity` plugin.
3. Drag & Drop the new xagithub widget and type your username and select how many entries do you want to show.
